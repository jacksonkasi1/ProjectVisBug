VisBug Smart Tracking Demo

This is a test page for the new VisBug "Smart Tracking" features.

Instructions:

  1. VisBug is loaded automatically. You should see the toolbar (press / if hidden).
  2. Select this text or any element.
  3. Change its color or typography using VisBug tools.
  4. Observe the neon "Changes" badge appear in the overlay.
  5. Press Cmd+Z to Undo, Cmd+Shift+Z to Redo.
  6. Click the badge to copy the code report.
